Unstable Unicorns Turn Tracker

Made by anwersaad0

Add a New Player

A maximum of 8 players will be visible on the board.



Name the Most Colorful Player

Their border will be highlighted pink to indicate that it is their turn.



Reset All Stables?

Players will not be removed, but all stables will be emptied.

Reset All Players and Cards?

Everything on the board will be removed.

Info

Main UI
Use 'Add Player' to add a player to the board
Use 'Start Game' to set which player will go first
'Reset Game' can be used to reset every player's stable
'Reset All' will clear the entire board
Player UI
'Add Card' can be used to add a card to a specific player's stable
'My Turn' will set the current turn to the selected player
'Remove Player' removes the selected player
Card UI
Enter the card name in the left text box
Enter the card effect in the right text box or leave blank if the card has no effect
'Submit Card' locks the name and description and adds the card to the player's stable
'Delete Card' removes the selected card from the player's stable
